# Smart Waste Tracking System ♻️

A web-based application to monitor and manage waste collection efficiently.

## 🚀 Features
- User authentication (Login/Signup)
- Waste tracking system
- Admin panel for monitoring
- Real-time data handling using MySQL

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
